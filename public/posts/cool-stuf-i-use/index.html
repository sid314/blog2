Yazi: A ranger replacement⌗

Yazi is a really fast TUI file manager, with support for plugins.

Yazi plugins I Use⌗

  • Reledia/glow
  • Rolv-Apneseth/starship
  • yazi-rs/plugins:full-border
  • KKV9/compress

Topgrade: Convenient way to manage your package manager⌗

Topgrade is a cli program to upgrade all your packages installed though pacman, flatpak, cargo, etc through one command. It also upgrades container images, git repositories, neovim plugins and much more.

Dua-cli: An NCDU replacement⌗

Dua-cli is a way faster ncdu replacement.

Gtrash: Better trash command⌗

Gtrash is a much better alternative to rm and trash-cli.

Bottom: Better Htop⌗

Bottom is much prettier htop replacement.

Gum: Make prettier shell scripts⌗

Gum provides tools to simplify writing pretty shell scripts.

Eza: Prettier ls⌗

Eza is a prettier, more colorful replacement for ls with lots of new features.
